# BattleCity
----
Cross-platform remake of BattleCity (Namco, 1985) with original Tanks AI and game mechanics

# Build
----
```console
git clone https://github.com/PfAndrey/BattleCity.git --recursive  
cd BattleCity  
mkdir build && cd build  
cmake ..  
cmake --build . --config Release  
./BattleCity 
```
